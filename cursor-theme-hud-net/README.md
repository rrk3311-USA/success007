## Installation

### From Cursor Marketplace

1. Open Cursor
2. Go to Extensions (Cmd+Shift+X / Ctrl+Shift+X)
3. Search for "HUD Net Core"
4. Click Install

### Manual Installation

1. Clone or download this repository
2. Copy the folder to your Cursor extensions directory:
   - **macOS**: `~/.cursor/extensions/`
   - **Windows**: `%USERPROFILE%\.cursor\extensions\`
   - **Linux**: `~/.cursor/extensions/`
3. Restart Cursor
4. Go to Settings > Color Theme and select "HUD Net Core"

## Customization

You can customize the theme by editing `themes/hud-net-core-color-theme.json`. The theme follows VS Code's color theme schema.

## Recommended Settings

For the best experience, consider these settings:

```json
{
  "editor.fontSize": 14,
  "editor.fontFamily": "'Fira Code', 'JetBrains Mono', monospace",
  "editor.fontLigatures": true,
  "editor.cursorBlinking": "smooth",
  "editor.cursorSmoothCaretAnimation": "on",
  "workbench.colorTheme": "HUD Net Core"
}
```

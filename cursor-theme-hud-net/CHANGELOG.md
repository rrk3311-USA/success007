# Change Log

All notable changes to the "HUD Net Core Theme" extension will be documented in this file.

## [1.0.0] - 2025-01-25

### Added
- Initial release of HUD Net Core Theme
- Dark base color scheme (#0a0e1a)
- Neon cyan accent color (#00d4ff)
- Complete syntax highlighting for all major languages
- Optimized color palette for code readability
- Dashboard-inspired UI elements
- High contrast design for accessibility
- Terminal color scheme matching the theme
- Git decoration colors
- Diff editor colors
- Editor widget styling

### Features
- Full VS Code/Cursor theme support
- All UI elements themed consistently
- Syntax highlighting for:
  - JavaScript/TypeScript
  - Python
  - HTML/CSS
  - JSON
  - Markdown
  - And more...

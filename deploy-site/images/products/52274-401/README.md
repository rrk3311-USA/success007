# Women's Balance Images (SKU: 52274-401)

Place all 12 product images in this folder.

## Naming Convention

Name images with numbers for ordering:
- `01.jpg` (main/hero image)
- `02.jpg`
- `03.jpg`
- ... up to `12.jpg`

Or use descriptive names:
- `main.jpg`
- `back.jpg`
- `ingredients.jpg`
- `lifestyle-1.jpg`
- etc.

Images will be sorted alphabetically when displayed.

Supported formats: JPG, JPEG, PNG, GIF, WEBP

# Product Images

Place product images in this folder.

## Naming Convention

Name your product images using the SKU number:
- `52274-401.jpg` for SKU 52274-401
- `[SKU].jpg` or `[SKU].png`

## Current Products

- **SKU 52274-401**: Women's Balance supplement
  - Place image as: `52274-401.jpg` or `52274-401.png`

The images will be accessible at: `http://localhost:3001/images/products/[filename]`
